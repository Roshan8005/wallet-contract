# Security Policy

Responsible disclosure
- If you discover a security vulnerability, please report it privately to: security@roshansah.dev
- We aim to respond within 72 hours and coordinate a responsible disclosure timeline.

What to include in a report
- Description of the issue and impact
- Steps to reproduce and PoC (if available)
- Affected version(s) or commit hash
- Your contact details and preferred communication method
- Optionally, a PGP public key for encrypted messages: REMOVE_IF_NOT_USED

Public disclosure
- Do not publicly disclose vulnerabilities until we've had time to fix them together.

Severity & timeline
- We'll acknowledge receipt within 72 hours.
- For critical vulnerabilities, we will prioritize fixes and coordinate patch and disclosure timeline.

Rewards or recognition
- We do not currently offer monetary bounties. We will credit reports in release notes unless you prefer to remain anonymous.

Emergency contact
- For emergency outreach, email: security@roshansah.dev (use PGP if using the provided PGP key).
